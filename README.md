# Snake Water Gun Game

A simple Python implementation of the classic Snake, Water, Gun game. This is a beginner-friendly project that demonstrates the use of Python concepts such as user input, conditional statements, loops, and the random module.

## How the Game Works

* Snake drinks Water → Snake wins
* Water douses Gun → Water wins
* Gun kills Snake → Gun wins
* If both players choose the same option, the result is a tie.

## Features

* Interactive command-line gameplay
* Random computer choices
* Win, lose, and tie detection
* Easy-to-understand Python code for beginners

## Technologies Used

* Python 3

## Learning Objectives

This project helps beginners practice:

* Variables and data types
* User input with `input()`
* Conditional statements (`if`, `elif`, `else`)
* Random number generation using the `random` module
* Basic game logic implementation

## How to Run

1. Clone the repository.
2. Run the Python file:

   ```bash
   python snake_water_gun.py
   ```
3. Enter your choice: `snake`, `water`, or `gun`.
4. Enjoy the game!
